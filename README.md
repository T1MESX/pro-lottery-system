Lottery Pro 是一款纯前端（SPA）、开箱即用、**极致安全且具备极高视觉表现力**的离线抽奖终端。V2 版本经历了彻底的底层重构，引入了 CSPRNG 密码学随机引擎、Canvas 动态海报生成以及全方位的防灾备系统，专为企业年会、展会活动、沙龙聚会打造。

## ✨ 核心特性 (V2 全新升级)

* 🛡️ **密码学级公平摇奖 (CSPRNG)**：抛弃了浏览器传统的 `Math.random()`，采用 Web Crypto API 获取底层硬件熵源，结合拒绝采样（Rejection Sampling）彻底消除取模偏差，确保绝对的统计学均匀性。
* 🎨 **Canvas 动态海报引擎**：内置工业级图片渲染引擎。一键自动生成带有活动大标题、智能排版对齐、防重叠换行的高清中奖名单海报（`.png`），支持直接导出朋友圈宣发。
* ⚡ **防视觉抖动排版 (Anti-Thrashing)**：采用 `tabular-nums` 等宽数字与动态 `ch` 单位锁宽技术，即便一次性抽取上百人、数字高速滚动，界面也绝对稳如泰山，零闪烁。
* 💾 **100% 断电灾备系统**：所有抽奖状态、中奖名单、奖品图片、星级评定均通过离线 `localStorage` 进行亚秒级快照。刷新、意外关机、断网皆不丢失任何数据。
* 🖼️ **沉浸式多媒体交互**：
  * 支持自定义奖品图片上传与智能防拉伸大图预览。
  * 独立的中奖庆祝弹窗，支持 1-5 星级特效与全屏 Confetti（纸屑）粒子动画。
  * 纯本地音频引擎，支持批量载入离线 MP3 歌单。
* 🔀 **双向作废与审计追溯**：发现作弊或缺席？一键将中奖记录作废，名额自动重返候选池；支持一键恢复。全量操作（配置修改、作废记录、中奖结果）均生成带时间戳的审计日志，支持 `.csv` 导出查账。
* 🌗 **双语与影院模式**：原生支持中文/English 一键无缝切换；内置基于深色模式的“影院模式（Cinema Mode）”，完美适应大屏幕投影环境。

## 🚀 极速上手 (Zero Config)

本项目为**零依赖**的纯前端单文件项目，无需安装 Node.js，无需配置任何服务器。

1. 克隆或下载本仓库代码到本地。
2. 双击打开 `index.html`（建议使用最新的 Chrome 或 Edge 浏览器）。
3. 如果需要默认背景音乐，请将您的音乐文件重命名为 `bgm.mp3` 并放置在 `index.html` 同级目录下；或直接在系统右上角点击 `[📂 本地音乐]` 动态加载。

## ⌨️ 快捷键指南 (Hotkeys)

为方便现场导播操作，系统全局集成了物理防误触热键：

| 操作 | 快捷键 | 说明 |
| :--- | :--- | :--- |
| **启停摇奖** | `Space (空格)` | 随时启动或停止号码滚动 (弹窗开启时自动锁定防误触) |
| **关闭弹窗** | `Enter (回车)` | 专门用于关闭“中奖结果”弹窗 |
| **关闭预览** | `Esc` | 专门用于关闭候选池预览、大图预览等次要弹窗 |
| **深色模式** | `D` | 快速切换明亮/影院模式 |
| **全屏切换** | `F` | 快速进入或退出浏览器全屏 |
| **音乐开关** | `M` | 快速播放或暂停背景音乐 |

## 📊 数据规则说明

* **支持的输入方式**：候选池支持 `Min` 到 `Max` 的范围生成。
* **排除名单格式**：支持单个剔除（如 `4, 15`）和范围剔除（如 `8-12`），支持换行输入。内置强迫症级防呆验证，自动过滤 `0`、负数及不合规格式。
* **不重复模式**：默认开启，已被抽中的号码（或被作废的号码）会自动从底层候选池中剔除，右上角日志将实时反馈真实的 `余量/总量`。
