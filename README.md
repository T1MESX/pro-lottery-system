# 🎬 专业级系统抽奖终端 (Pro Lottery System)

一个基于原生前端技术栈构建的专业级、高安全性、带物理特效的网页端抽奖系统。专为中大型活动、年会及协会活动设计。

## ✨ 核心特性 (Features)

* **🔒 绝对公平的密码学安全**：摒弃传统的 `Math.random()`，底层采用 Web Crypto API (`window.crypto.getRandomValues`)，直接调用系统底层硬件熵源（CSPRNG），并在代码层面消除了取模偏差（Modulo Bias），确保绝对的统计学均匀性。
* **📊 极客级数据透视面板**：
    * 实时监控：剩余奖池/总量、全样本理论均值、已抽结果均值。
    * 高级校验：**离散方差**校验分布均匀度，**奇偶天平**监控分布特征，**区间热力**（低/中/高位区）实时追踪随机落点。
* **🎮 原生高性能物理引擎**：无需任何第三方依赖（零 npm 包），纯原生 Canvas + Web Animations API 打造全屏 3D 彩带物理瀑布流、鼠标粒子交互与弹窗阻尼震动特效。
* **📑 商业级审计日志导出**：内置结构化日志管理，支持一键将抽奖结果、配置变更、系统提示等操作导出为 Excel 兼容的格式化 `.csv` 文件，方便活动复盘与合规存档。
* **🛠 强大的防呆交互配置**：支持 `a-b` 号段快速排除、自动去重排序、小数与异常字符拦截、以及防误触的“全域空格键启停”控制。

## 🚀 快速开始 (Quick Start)

本项目为纯静态前端单页面应用（SPA），**零依赖，即开即用**。

1. 克隆本仓库到本地：
   \`\`\`bash
   git clone https://github.com/T1MESX/pro-lottery-system.git
   \`\`\`
2. 直接使用现代浏览器（推荐 Chrome / Edge / Safari）双击打开 `Lottery-pro.html` 即可运行。

## 💡 使用指南 (Usage)

1. **基础配置**：在左侧设置区输入起始号码、截止号码、单次抽取人数。
2. **排除名单**：支持独立数字（如 `4, 13`）或号段范围（如 `8-12`），失去焦点后会自动解析并格式化。
3. **星级标记**：点击星星可为当前奖项赋予 `1~5` 星级。当触发 `5星` 满星奖项时，将解锁史诗级全屏物理特效。
4. **奖池透视**：双击蓝色数字滚动区，可呼出当前剩余有效奖池的实时快照。
5. **审计导出**：抽奖结束后，点击右下角的 `[ ↓ 导出格式化日志 (.csv) ]`，即可获取完整活动记录。
