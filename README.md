Lottery Pro 是一款**纯前端（SPA）、零依赖、开箱即用**的专业级活动抽奖系统。它被设计用于解决企业年会、展会活动及大型发布会中“暗箱操作”的信任危机，同时提供导播级的沉浸式多媒体交互体验。

无需繁琐的数据库配置，无需安装任何服务端环境。**下载单 `HTML` 文件，双击即可运行。**

## ✨ 核心特性 (Features)

### 🛡️ 绝对公平的抽奖引擎
- **摒弃 `Math.random()`**：底层采用密码学标准的安全引擎 **Web Crypto API (CSPRNG)**，从硬件层面提取高熵随机数。
- **拒绝采样算法 (Rejection Sampling)**：彻底消除大基数下的“取模偏差（Modulo Bias）”，确保每一个人/号码被抽中的概率具备完美的统计学均匀性。
- **安全审计哈希**：每一轮抽奖均会提取 16 字节硬件随机噪声，生成独立的 `SHA-256` 级别安全种子哈希（Seed Hash），并支持随日志导出，满足企业级防篡改公证需求。

### 🎭 导播级沉浸式交互
- **双模无缝热切**：一键在中英文 (i18n) 与 影院/明亮模式间热重载，不丢失任何状态。
- **三轨独立音频引擎**：支持 BGM（背景乐）、Rolling SFX（滚动音效）、Success SFX（开奖音效）。内嵌带呼吸感的**无级音量悬浮气泡**（自动保存用户音量偏好）。
- **全局拖拽网 (Drag & Drop)**：
  - 🖼️ 拖入图片 -> 自动上屏并应用智能压缩。
  - 📄 拖入名单 (`.xlsx`, `.csv`, `.txt`) -> 瞬间拦截、智能分割并静默去重。
  - 🎵 拖入音频 (`.mp3`, `.flac`) -> 无缝热替换当前音轨。

### ⏪ 时光倒流与状态闭环
- **因果律回滚**：在日志中点击“作废”，被废弃的奖项页签不仅会恢复可选状态，还会瞬间将奖品名、星级、大图**逆向重载回主控制台**，宛如时光倒流。
- **全自动缓存与灾备**：每一次配置变更和抽取操作，都会被实时写入浏览器 `LocalStorage`，即使意外断电、刷新网页，进度也能 100% 无损恢复。
- **状态快照 (State Snapshot)**：支持将当前所有的黑名单、奖项预设、历史记录和内嵌图片打包为独立的 `.json` 文件导出，一键实现**跨电脑接力与现场换机**。

## 🚀 极速上手 (Quick Start)

### 1. 运行方式
无需部署，直接使用现代浏览器（推荐 Chrome / Edge）打开 html文件 即可。

### 2. 名单导入指南
系统支持两种抽取模式：**数字区间抽取** 与 **名单模式**。
若要导入名单，请点击 `[📥 导入名单]` 或直接将文件拖入巨大的蓝色摇奖区：
- **Excel (.xlsx / .xls)**：系统严格且**仅解析 Sheet1 的 A列**。⚠️ 请务必删除表头（第一行），以免标题中奖！
- **TXT / CSV**：支持换行、空格、全半角逗号作为分隔符，引擎将进行完美智能分割。
- *注：系统在载入名单时，会自动进行全局扫描并剔除重复项。*

### 3. 音频资源盲埋 (可选)
为了获得极佳的晚会氛围，你只需在 html文件 的**同一文件夹目录下**放入以下三个文件，系统启动时将**全自动静默加载**：
- `bgm.mp3`：全局循环背景音乐。
- `rolling.mp3`：按下空格键开始滚动时的音效（如机械齿轮、心跳加速）。
- `success.mp3`：停止滚动、开奖弹窗弹出瞬间的音效（如礼花绽放、金币声）。
*(若未放置文件，系统依然正常运行，控制台亦支持直接拖拽 `.mp3` / `.flac` 热替换)*。

## 📊 数据管理与导出

系统右下角提供完整的数据全生命周期管理：
- **🎨 生成海报**：一键将最近 N 轮的有效中奖记录，生成排版精美的高清 `PNG` 公示海报。
- **↓ 导出日志**：将所有历史记录（含有效、作废追踪、奖项明细、硬件哈希、余量快照）导出为符合规范的 `CSV` 审计报表。
