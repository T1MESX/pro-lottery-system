公平性与算法声明 ⚖️： 本终端程序完全在本地浏览器环境中运行，无任何服务器后端交互与暗箱操作可能。核心抽奖逻辑抛弃了常规的 Math.random() 伪随机生成器，采用 Web Crypto API 提供的 密码学安全伪随机数生成器 (CSPRNG)。

👉 生成原理： 系统直接调用操作系统底层硬件（如CPU热噪声、鼠标微观轨迹等）的混合熵源作为种子，通过 window.crypto.getRandomValues() 提取出极高信息熵的随机字节，并在代码层面结合了防取模偏差（Modulo Bias）的安全重试算法。该机制确保了无论设定的奖池多大，每一个有效号码被抽中的概率均具备极其严密的统计学均匀性与绝对的不可预测性。
